# Mini RPG

## Sobre o projeto

Mini RPG é um projeto desenvolvido com JavaScript puro com foco no aprendizado de desenvolvimento Front-End.

O projeto não tem como objetivo principal criar um jogo completo, mas sim servir como ambiente de estudo para aplicar conceitos aprendidos durante a jornada de desenvolvimento.

A cada novo conteúdo estudado, novas funcionalidades serão adicionadas ao projeto.

---

## Objetivos

- Praticar JavaScript.
- Evoluir continuamente o mesmo projeto.
- Aprender organização de código.
- Desenvolver boas práticas.
- Simular um ambiente de desenvolvimento profissional.
